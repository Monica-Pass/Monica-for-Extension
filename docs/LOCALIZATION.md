# 本地化支持范围

Monica for Extension 首发界面语言为 **zh-CN（简体中文）**。管理页、Popup、保存提示、Passkey 提示、错误恢复文案和帮助文档均以简体中文作为当前受支持的产品语言。

浏览器商店可见的名称、说明和工具栏标题通过 Manifest V3 `_locales/zh_CN/messages.json` 提供，Manifest 本身不再硬编码展示文案。新增语言时必须同时完成以下内容后才可声明支持：

1. `_locales/<locale>/messages.json` 的商店和工具栏文案；
2. 管理页、Popup、Content Script 提示及后台错误的完整翻译；
3. 日期、数字和复数规则的对应区域测试；
4. 该语言下的 200% 缩放、键盘、屏幕阅读器和商店截图验收。

字段识别并不等同于界面语言支持。当前自动填充可保守识别部分英文和中文字段，WebDAV 也会保留 Android 未知字段，但这不代表英文界面已经完成。未达到上述四项门槛前，不在商店文案中宣称其他完整界面语言。
